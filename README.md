# Apprendre jQuery

## Pré-requis

Ce cours est destiné à des personnes ayant déja de bonnes connaissances HTML/CSS et quelques bases de JavaScript.
Nous ne traiterons pas de toutes les méthodes de jQuery, mais des plus utilisées.
